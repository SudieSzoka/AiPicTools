图片居中缩放工具

上传图片，自动等比例缩放并居中放置在指定大小的画布中

拖拽图片到此处或点击选择
支持 PNG、JPG、WebP、GIF、BMP、AVIF，最多 50 张，每张不超过 25 MB

设置参数

尺寸必须是 50-2000 的整数；导出画布最大 2000×2000，图片会等比例缩小并保持完整可见。

50%

使用说明：

  1. 设置最大尺寸（缩放后图片的宽或高最大值）
  2. 设置画布的宽度和高度
  3. 选择背景类型：
    • 透明背景：保持画布透明
    • 纯色背景：选择单一颜色作为背景
    • 渐变背景：选择起始和结束颜色，设置渐变方向
    • 图片背景：上传背景图片，可调节透明度
  4. 拖拽图片到上传区域或点击选择图片文件
  5. 处理后的图片将实时显示在下方，点击下载按钮保存
  6. 多张图片时可使用批量下载功能
