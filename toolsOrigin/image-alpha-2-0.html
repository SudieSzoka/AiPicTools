深色背景透明化

将 RGB 三个通道都不高于所设阈值的像素变为透明，适合处理黑色或近黑色纯背景。 结果按原始分辨率导出为 PNG，不会上传到服务器。

透明化参数

0 为硬边；1-64 会让阈值附近像素逐渐透明。

仅当像素同时满足 R ≤ 红色上限、G ≤ 绿色上限、B ≤ 蓝色上限时，才会完全透明。
